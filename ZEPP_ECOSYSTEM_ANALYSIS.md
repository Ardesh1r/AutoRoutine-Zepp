## 📱 Zepp OS Platform

### What is Zepp OS?
A lightweight operating system for smartwatches and wearables designed around health management and user experience. It powers Amazfit watches and supports third-party Mini Programs.

### Key Characteristics
- **Language**: JavaScript (ES6+)
- **Framework**: Lightweight, event-driven
- **Architecture**: Device App + Side App (optional)
- **API Level**: Currently 4.2 (latest)
- **Minimum**: API Level 3.0
- **Memory**: 32-256MB per device
- **Storage**: 50KB-2MB per app

### Supported Devices
**Current Generation** (Zepp OS 4.x):
- Amazfit Active 2
- Amazfit T-Rex 3
- Amazfit Balance
- Amazfit Bip 5
- Amazfit GTS 4
- Amazfit GTR 4
- Amazfit GTR 4 Pro

**Previous Generation** (Zepp OS 3.x):
- Amazfit GTS 3
- Amazfit GTR 3
- Amazfit Bip U Pro
- Amazfit Band 7
- And 20+ more models

**Total Compatible Devices**: 50+
